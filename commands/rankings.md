Analyze current business school rankings and Carey's competitive position.

## Instructions

1. Load the `competitive-intelligence` skill for rankings frameworks

2. Research current rankings from:
   - US News & World Report
   - Bloomberg Businessweek
   - Financial Times
   - QS World University Rankings
   - Poets & Quants

3. For each ranking, document:
   - Carey's current position
   - Year-over-year change
   - Peer school positions
   - Methodology highlights
   - Key metrics affecting rank

4. Create analysis including:
   - Rankings summary table
   - Trend analysis (3-5 year view)
   - Peer comparison
   - Methodology insights
   - Improvement opportunities

5. Identify actionable recommendations for rankings improvement

## Output
Save to: `./outputs/rankings/Rankings_Analysis_[Date].md`
